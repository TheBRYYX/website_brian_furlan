E-Portfolio

Die Lehre als Informatiker Applikationsentwickler beinhaltet im ersten Schuljahr das Modul 431 auch "IT-Aufträge selbständig durchführen" genannt. Eine dieser Noten kommt vom E-Portfolio. Das E-Portfolio ist eine Dokumentation, die man über das ganze Semester führt. Sie beinhaltet verschiedene Schritte oder Themen, die man einbinden muss. Ein paar Beispiele, über die man in dieser Dokumentation schreiben muss, sind die Projektmethode IPERKA und wie sie unterteilt ist oder auch die ALPEN-Methode.

Java Übung

Eine weitere Aufgabe, die ich erledigt habe, hat mit Java zu tun. Wie oben bereits erwähnt, verwende ich Java in meinem Betrieb. Die Lehrmeister geben uns Aufgaben, die wir erledigen können. Eine dieser Aufgaben, die ich programmiert habe, ist ein Ticketautomat. Dieser Ticketautomat beinhaltet alle Postleitzahlen der Schweiz. Man kann sich einfach ein Ticket erstellen, mit dem Zielort. Anschliessend kann man weitere Funktionen auswählen wie ein Rückfahrtticket lösen oder 1. Klasse fahren usw.
